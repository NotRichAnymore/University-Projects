The purpose of the website is to create and show a visual output as a result of the activities which I have pursued as a part of the Essential Skills module for the Continued Professional Development part of the module. It was essential that the website was written in HTML and CSS in notepad++ to ensure that a good understanding of web layout techniques as well as HTML knowledge.

CPD (Continuing Professional Development) is a word used in conjunction with educational activities which are said to strengthen and progress the abilities of the recipient. It allows for the flow of learning to be constant and active rather than dormant and stale. A range of activities such as : workshops, e-learning programs and events are used to improve and encourage professional development.
